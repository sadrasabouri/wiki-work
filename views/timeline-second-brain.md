---

The "second brain" concept entered this project not from a meeting but from Tiago Forte's pre-LLM methodology, which established that humans need an external digital memory to handle the capture-organize-distill-express cycle. The LLM-wiki pattern (Karpathy) was the trigger that made automation of that cycle feel tractable. Within this project, the concept evolved from an implicit analogy (intermediate summaries ≈ Intermediate Packets, May 1) to a named predecessor (LLM-Wiki Related Works, May 4) to a positioned layer in an explicit intellectual lineage (May 5). Where it stands now: BASB provides the "why" (external memory as cognitive infrastructure); Karpathy provided the "how" for personal use; this project provides the "how" for teams and organizations, with semi-private mashing as the enabling mechanism that makes organizational scope safe.
