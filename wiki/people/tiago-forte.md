# Tiago Forte

Creator of the **Building a Second Brain** (BASB) methodology and founder of Forte Labs. Author of the CODE method (Capture, Organize, Distill, Express) and the PARA organizing system. The BASB framework is the pre-LLM intellectual precedent for the LLM-wiki pattern — establishing the same Capture → Organize → Distill → Express workflow that LLM agents like Claude now partially automate.

Forte is not directly involved in the [[wiki-kb-project|Wiki KB Project]], but his framework provides the historical grounding for why structured PKM matters and what the ideal state looks like. His "Intermediate Packets" concept anticipates the Wiki KB Project's "efficiency via intermediate summaries" contribution.

## Key Contributions

- **CODE method** — four-step PKM workflow; maps directly to LLM-wiki pipeline components (see [[second-brain-code-method]])
- **PARA method** — organizing by actionability (Projects, Areas, Resources, Archive); contrasts with wiki's organizing by conceptual relation
- **Progressive Summarization** — multi-layer note distillation; equivalent to multi-resolution views
- **Intermediate Packets** — reusable work chunks; structurally equivalent to intermediate summaries
